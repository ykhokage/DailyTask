# DailyTasks

DailyTasks —  Android-приложение ежедневник.



### Базовый функционал
- экран со списком задач на выбранный день
- выбор даты 
- почасовая сетка на 24 часа
- отображение задач в каждом часовом интервале, который они пересекают
- экран подробного описания задачи
- возможность редактирования и удаления задач

### Дополнительный функционал
- экран создания новой задачи
- локальное хранение через Room
- первичная загрузка demo-данных из `assets/tasks.json`
- автоматическое обновление расписания после создания новой задачи
- unit-тесты для ключевой бизнес-логики

## Стек
- Kotlin
- Jetpack Compose
- MVVM
- Room
- Navigation Compose
- Coroutines / Flow

## Архитектура
Проект разделён на слои:
- `data` — Room, JSON, репозиторий, мапперы
- `domain` — модели, use case, сервис построения расписания
- `presentation` — ViewModel и экраны
- `main` — DI-контейнер приложения и корневой NavHost

## Ключевое решение по отображению расписания
Сервис `DefaultScheduleService` строит 24 слота дня (`00:00-01:00` ... `23:00-00:00`) и помещает задачу во все слоты, с которыми пересекается её временной диапазон. Это корректно работает и для длинных задач, и для задач, переходящих через полночь.

## Как запустить
1. Открыть проект в Android Studio.
2. Дождаться Gradle Sync.
3. Запустить приложение на эмуляторе или устройстве с Android 8+.

## Примечание
В проекте UI полностью реализован на Jetpack Compose.
